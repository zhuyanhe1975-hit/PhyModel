# models/

本目录包含 ER15-1400 的模型文件（用于 PerfOpt v0 的 MuJoCo CPU 仿真与对比）。

## 文件

- `er15-1400.mjcf.xml`：MuJoCo MJCF（主线使用）
- `ER15-1400.urdf`：URDF（备用/对照；v0 主线不依赖）
- `*.STL`：网格（可视化/碰撞几何）

## 重要注意（踩坑总结）

- 跟踪误差异常大、怎么调 PID 都不稳定时，优先排查 **自碰撞/相邻连接体碰撞**。
  - 接触约束会引入强非线性外力，误导控制与优化（看起来像动力学/控制错了，其实是碰撞在“打架”）。
  - v0 默认优先保证“无自碰可控环境”跑通主线；后续再逐步恢复必要碰撞/接触。

## 快速验证

验证 MJCF 可加载（并打印关节/限位等关键信息）：

```bash
python scripts/validate_mjcf.py --mjcf models/er15-1400.mjcf.xml
```
